### webpack Boilerplate 
webpack and KSS (Knyle Style Sheet) boilerplate.


### Setup 
```bash
1. Make sure Node and NPM are installed.
2. git clone https://github.com/vpasq/webpack-kss-boilerplate.git
3. cd webpack-kss-boilerplate
4. npm install
5. npm run build
6. npm run kss
7. open docs/index.html
```
